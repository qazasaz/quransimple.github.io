modifications to the original xml file:
1) an attribute (trans) has been added to each aya tag which holds farsi translation.
the original translation file has been downloaded from tanzil translation page: http://tanzil.net/trans/
2) an attribute (roots) has been added to each aya tag which holds the root of each word of quran.
these roots were added to this file using zekr (an open-source quranic application) metadata file.
you can download zekr from http://zekr.org
zekr metadata file also has been modified before being used.
3) another version of zekr metadata file in xml structure has been created and added to main xml file.

below is the copyright block of zekr metadata file (adjustment notes included) and also copyright block of farsi translation files:
